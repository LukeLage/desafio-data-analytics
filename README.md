# Projeto Final — Trilha de Análise de Dados
## Projeto Desenvolve Itabira

Este projeto foi desenvolvido como trabalho de finalização do **Projeto Desenvolve Itabira**, com a trilha utilizada para a finalização sendo a de **Data Analytics**, com o objetivo de analisar dados de vendas de uma loja utilizando Python e ferramentas de visualização.

---

## 📦 Dependências

Para rodar este projeto, você precisará das seguintes bibliotecas:

```plaintext
streamlit
pandas
matplotlib
sqlite3
plotly
```

Você pode instalar as dependências com o comando:

```bash
pip install streamlit pandas matplotlib plotly
```
> O `sqlite3` já faz parte do Python padrão.

## 🚀 Como rodar o projeto

1. Clone o repositório:
```
bash
git clone https://github.com/seu-usuario/seu-projeto.git
```
2. Instale as dependências.
3. Execute o app Streamlit:

```bash
streamlit run app.py
```

## 📊 Principais Insights

### Perfil dos Clientes
- **Mulheres:** 59.482 clientes (**59,8%**)
- **Homens:** 39.975 clientes (**40,2%**)

### Estação do Ano com Mais Vendas
- **Verão** é a estação com o maior volume de vendas.

### Categoria Mais Vendida
- **Roupas:** Mais de **30.000** vendas
- **Cosméticos:** Aproximadamente **10.500** vendas

### Perfil Etário dos Compradores
- **Média de idade:** 43 anos
- **Moda (idade mais frequente):** 37 anos

### Previsão de Mercado
- Segundo predições do Power BI, **há alta possibilidade de queda de lucros e vendas** nos próximos períodos.
